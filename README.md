# Vendas-produtos-globais

Análise de dados de um datset que contém informações sobre as vendas de produtos de uma empresa que atua no mercado global, comercializando mercadorias tanto em lojas físicas quanto em lojas online. 

O conjunto de dados é composto por três tabelas:

events.csv — vendas ao longo de vários anos;
products.csv — categorias de produtos e seus respectivos códigos;
countries.csv — países, regiões e seus respectivos códigos.

O objetivo principal é limpar os dados, realizar uma análise e descobrir insights valiosos sobre as operações da empresa.

Utilizei o google colab para realizar todas as operações de análise desde o merge entre as tabelas até a construção dos gráficos e o relatório final com insights e sugestões.

Todo o processo está documentado e dividido com títulos e explicações sobre cada etapa que realizei.
